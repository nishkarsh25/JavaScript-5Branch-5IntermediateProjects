# Filter Product

Welcome to the Filter Product project! This project provides a user-friendly interface for filtering products based on search queries and categories. By leveraging HTML, CSS (using Tailwind CSS), and JavaScript, this project offers an intuitive way for users to explore and find products efficiently.

## Key Features

### 1. Product Filtering
- Users can filter products based on search queries entered into the search input field.
- Products can also be filtered based on specific categories selected using category buttons.

### 2. Dynamic Product Display
- The product list dynamically updates in real-time as users input search queries or change category selections.
- Hidden items are removed from the display, ensuring a seamless and responsive user experience.

### 3. Responsive Design
- The interface is designed to be responsive, ensuring optimal viewing and interaction across various devices and screen sizes.

## Technologies Used

- **HTML**: Provides the structure and content of the product filtering interface.
- **CSS (with Tailwind CSS)**: Offers styling for the interface components, ensuring a visually appealing and consistent design.
- **JavaScript**: Implements the logic for filtering products based on user input and category selections, enhancing interactivity and functionality.

## How to Use

Using the Filter Product interface is simple:

1. **Search**: Enter a search query into the search input field to filter products based on matching titles.
2. **Category Selection**: Click on category buttons to filter products based on specific categories.
3. **View Products**: Explore the filtered product list displayed below the search and category sections.
